# More Robits for Mekanism

[Modrinth](https://modrinth.com/project/more-robits-for-mekanism)

This mod adds a few simple skins to the Mekanism Robit.

They come in solid Red, Orange, Yellow, Green, Cyan, Blue, Purple and Pink colors.
